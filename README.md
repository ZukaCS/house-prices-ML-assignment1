# house-prices-ML-assignment1
## კონკურსის მიმოხილვა

House Prices კონკურსის მიზანია სახლების ფასების პროგნოზირება სხვადასხვა მოცემული მახასიათებლების საფუძველზე. ამოცანა ფასდება (Root-Mean-Squared-Error)-ის მიხედვით predicted და რეალურ ფასებს შორის.
