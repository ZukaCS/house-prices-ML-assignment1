## მონაცემთა გაწმენდა (Data cleaning)
### 1. გამოტოვებული მნიშვნელობების სვეტების გაქრობა.
    
მოვძებნე ისეთი სვეტები რომლებშიც ძალიან დიდი პროპორციით იყო missing-value/null -ები და ამოვშალე

ასეთი სვეტები იყო: 'PoolQC', 'MiscFeature', 'Alley', 'Fence', 'FireplaceQu', 'LotFrontage'. სურათის მიხედვით მათ ბევრი null მნიშვნელობები ჰქონდათ ამოტომ მათი შევსება უფრო მეტ ხმაურს შეიტანდა მოდელში და არ მოგვცემდა კარგ შედეგებს. ამ შემთხვევაში მოდელი ისწავლიდა არარსებულ კანონზომიერებებს და გვექნებოდა overfitted მოდელი.

ასევე ამოვშალე 'Id' სვეტი რადგან ეს უბრალოდ თითოეული სვეტის იდენთიფიკატორია რაც განსაკუთრებულ მნიშვნელობას არ ატარებს.
